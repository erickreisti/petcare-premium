# Petcare Premium 🐾

Uma moderna landing page para um Centro Veterinário de Excelência, construída com Next.js, React e Tailwind CSS v4.

## 🌟 Funcionalidades
- **Design Premium**: Interface sofisticada utilizando Glassmorphism, paleta de cores institucional (Azul Marinho e Laranja) e Soft Shadows.
- **Responsividade**: Layout 100% otimizado para celulares, tablets e desktops.
- **Animações**: Micro-interações e animações CSS (Reveal, Magnetic Hover).
- **SEO Otimizado**: Tags OpenGraph, title e meta descriptions configuradas para melhor indexação e compartilhamento.

## 🛠️ Tecnologias Utilizadas
- [Next.js](https://nextjs.org/) (App Router)
- [React](https://react.dev/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)

## 🚀 Como Executar Localmente

1. Instale as dependências do projeto:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Abra o navegador em [http://localhost:5173](http://localhost:5173) ou a porta indicada no terminal.

## 📄 Estrutura Principal
- `src/components/`: Componentes UI da Landing Page (`Header.tsx`, `Footer.tsx`, `Home.tsx`, `About.tsx`, `Prices.tsx`, e `Services.tsx`).
- `src/app/`: Configurações de layout global e metadados (`layout.tsx`, `page.tsx`).
- `public/images/`: Assets visuais e fotos.
